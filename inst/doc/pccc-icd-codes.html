Viewing ICD Codes

The ICD codes used to define the categories of CCCs have been hard-coded into the C++ backend of the pccc package. The get_codes function provides the end user a easy way to view all of the ICD codes as strings.

Make sure the library is loaded first

library(pccc)

Call get_codes function:

get_codes(9)   # ICD-9-CM
